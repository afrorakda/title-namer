# Title Namer

Part of the Fridge Combo project. A professional naming tool to standardize your files with the YYMMDD format and sequential numbering.

## 1 Tool / 1 Action

Generate OS-friendly filenames instantly. **Title Namer** automatically prefixes today's date (YYMMDD) and appends a sequence number to your project title, ensuring perfect order in your folders.

## Philosophy
* **1 Tool, 1 Action.**
* **No Login / No Ads / No Tracking**
* **Browser-only Privacy**

## How to Use
1. **Input:** Type your project or content name.
2. **Auto-Sanitize:** Spaces and OS-forbidden characters (`\ / : * ? " < > |`) are automatically converted to underscores (`_`) to ensure file compatibility.
3. **Sequence:** Use the [ − ] and [ + ] buttons to adjust the version number (01, 02, 03...).
4. **Copy:** Tap the generated name or the [ COPY FILENAME ] button to save it to your clipboard.

## Why use it?
Eliminates the "What's today's date?" pause and prevents broken filenames by cleaning your input in real-time. Perfect for designers, developers, and anyone who loves an organized folder.
